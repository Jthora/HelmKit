# Track D — Executable notebooks (`notebooks/`)

- **Status**: `ready`
- **Effort**: ~3–4 hours per notebook
- **Depends on**: Track C (each notebook usually pairs with a derivation)
- **Unblocks**: figure assets for Field Notes Vol. I (Track F)

---

## Goal

A set of small, deterministic Jupyter notebooks that:

1. Regenerate every figure used in `docs/derivations/` and Field Notes Vol. I.
2. Are CI-runnable (no external network, no proprietary data, pinned Python deps).
3. Export figures to `docs/field-notes/volume-1/figures/` as both `.png` (web) and `.pdf` (print).

## Why notebooks (not just plots)

- A reader who buys Field Notes Vol. I can clone the repo, run the notebook, and reproduce every figure. That is the falsifiability discipline applied to publication.
- The notebooks are also the citation hook for academic re-use: "Figure 4 generated by `notebooks/bifilar_near_field.ipynb` revision <SHA>."

## First notebook

**`notebooks/bifilar_near_field.ipynb`**

- Imports closed-form $\eta(r)$ from Track C derivation.
- Plots Figure 4 of Vol. I: $\eta$ vs $r/a$, log-log.
- Plots Figure 5 of Vol. I: field-component breakdown at $r = 5$ cm vs frequency 1 Hz – 1 MHz.
- Saves to `docs/field-notes/volume-1/figures/fig04_bifilar_enhancement.{png,pdf}` and `fig05_bifilar_freq_response.{png,pdf}`.

## Conventions

- Python 3.11+. Pinned in `notebooks/requirements.txt`.
- `numpy`, `scipy`, `matplotlib` only (no domain libs).
- Plot style: matplotlib default + `plt.style.use('seaborn-v0_8-whitegrid')`. Mono-friendly (project ships PDF).
- Every figure has a caption cell in markdown immediately above the plot cell.
- Every notebook ends with a "Reproducibility" cell printing `numpy`, `scipy`, `matplotlib` versions.

## CI hook (optional, can ship later)

`.github/workflows/notebooks.yml`:

```
- name: Execute notebooks
  run: |
    pip install -r notebooks/requirements.txt
    jupyter nbconvert --to notebook --execute notebooks/*.ipynb --inplace
- name: Verify no diff in figures
  run: git diff --exit-code docs/field-notes/volume-1/figures/
```

This catches the case where a derivation gets edited but the figure didn't get re-rendered.

## Acceptance criteria

1. `jupyter nbconvert --to notebook --execute <nb>` succeeds on a clean clone within 30s per notebook.
2. Output figures bit-identical (or within matplotlib's reasonable diff) across runs on same machine.
3. Each figure file is < 300 KB PNG / < 50 KB PDF (vector).
