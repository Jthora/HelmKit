# `docs/outreach/`

Outreach assets and scripts. Most of this is text-of-record for things that will be posted off-repo (TikTok, X, Ko-fi, email). Keeping them here means:

1. Versioned. We can see when copy changed and why.
2. Reviewable. Posts get the same scrutiny as docs.
3. Re-usable. A second TikTok / X cycle drafts from the previous one, not from scratch.
